## What I Learned

I started this project early because Data Structures and Algorithms felt difficult from the beginning.
The course required steady practice, and I had to learn while building at the same time.

## Key Topics

- Pointers and pointer-to-pointer usage
- Dynamic memory management
- Additional data structures beyond the basics
- Algorithm references and implementation practice
- Debugging when the first solution was wrong

## Main Insight

After several failed attempts in class, I eventually found a queue-based code generation algorithm that helped keep the binary tree balanced.
I did not reach every requirement perfectly during the exam, but I understood the idea much more deeply by the end.

## Project Notes

This project was a practical way to connect course theory with real implementation.
It also taught me that progress comes from repeated practice, careful debugging, and choosing the right structure for the problem.
